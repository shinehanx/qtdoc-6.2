Obsolete Members for Application

The following members of QML type Application are deprecated. They are provided to keep old source code working. We strongly advise against using them in new code.

Properties

  • active : bool (deprecated since 5.2)

Property Documentation

active : bool

This property is deprecated since 5.2. We strongly advise against using it in new code.

Returns whether the application is active. Use Qt.application.state == Qt.ApplicationActive instead
