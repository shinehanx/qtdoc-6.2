qt_no_entrypoint

Specifies to inhibit linking against Qt's entrypoint lib.

Note: This property is in technology preview and may change in future releases.

On certain platforms, Qt applications link against Qt's entrypoint lib by default. That library provides implementations of main (or WinMain).

On targets that must provide their own entry point, set the property qt_no_entrypoint to inhibit linking against Qt's entrypoint library.
