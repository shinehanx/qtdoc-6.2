Monolith Project Tutorial

The INTEGRITY applications can be monolith application projects, dynamic download application projects, or kernelspace projects. The monolith application is a stand-alone executable that you can run directly on a target device.

In this tutorial, we build an INTEGRITY monolith project for a Qt application. After following the steps in this tutorial, you will have the Qt for INTEGRITY development environment up and running.

The tutorial assumes you have the Qualcomm Snapdragon 8155P.

Tutorial content:

  • Installing Platform Dependencies
  • Build Qt with Ubuntu host
  • Build Qt with Windows 10 host
